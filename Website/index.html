About BOT 441

The aim of BOT 441 is to make a Discord bot primarily focused on esports that has all the features a user could want.

We are a small team of computer science students and this is our software engineering project.

Command List See it in action

Command List

!echo (string)

repeats anything user inputs

!live (streamname)

checks Twitch if user-input streamer is online and returns a link to that stream

!upcoming (lol / ow / dota2)

returns 4 upcoming matches for input game

!notification (enable)

sends a notification whenever a stream goes live
